# Vendor Performance Analysis Project

## Project Overview
This project analyzes vendor performance, sales trends, and inventory efficiency in the retail and wholesale industry.

The goal is to identify top-performing vendors, detect underperforming brands, and optimize pricing and inventory strategies using data analytics techniques.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Excel
- Jupyter Notebook

## Project Workflow

1. Define Business Problem
2. Explore Database Tables
3. Data Cleaning and Data Merging using SQL
4. Load aggregated dataset into Python (Jupyter Notebook)
5. Perform Exploratory Data Analysis (EDA)
6. Analyze research questions and interpret insights
7. Create an interactive Power BI dashboard
8. Generate final analytical report

## Key Business Questions
- Which vendors contribute the most to total sales?
- Which brands are underperforming?
- What is the impact of bulk purchasing on unit costs?
- Which vendors have slow inventory turnover?
- How does profit margin vary between high-performing and low-performing vendors?

## Key Insights
- Total Sales: $441.41M
- Total Purchase: $307.34M
- Gross Profit: $134.07M
- Profit Margin: 38.7%
- Unsold Inventory Capital: $2.71M

## Major Findings
- Top 10 vendors contribute 65.7% of total purchases.
- Bulk purchasing reduces unit costs significantly.
- Some vendors maintain high profit margins but low sales volume.
- Slow-moving inventory leads to $2.71M tied-up capital.

## Business Recommendations
- Optimize pricing strategies for low-sales brands.
- Diversify vendor partnerships to reduce dependency risks.
- Improve inventory turnover and inventory management.
- Strengthen marketing strategies for high-margin brands.
